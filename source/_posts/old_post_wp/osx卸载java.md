通过删除Java插件文件来卸载 Oracle Java
插件名字:
JavaAppletPlugin.plugin。

*要卸载 Java，必须具有管理员权限。*

* 单击位于停靠栏中的 Finder 图标
* 单击边栏中的应用程序选项卡
* 在搜索框中，输入 JavaAppletPlugin.plugin
* 此命令将查找 JavaAppletPlugin.plugin 文件
* 右键单击 JavaAppletPlugin.plugin，然后选择移到废纸篓
