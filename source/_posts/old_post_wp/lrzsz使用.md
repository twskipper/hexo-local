lrzsz
安装方法
centos
``` bash
yum install -y lrzsz
```
ubuntu:

``` bash
apt install -y lrzsz
```
使用方法
``` bash
sz file 将文件发送出来
rz 将文件传输都linux中去
```

*具体可以参见 rz -h 或者  sz -h*
